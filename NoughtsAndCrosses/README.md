# Noughts And Crosses
Exhaustive search for noughts and crosses (tic-tac-toe).

## Usage
`noughtsandcrosses` for full search

`noughtsandcrosses [position]` for a specific position
(assumes X is the first player)
<br></br>
### Example
`exhaust ..o.xoxox`
<pre>
| | |o|
| |x|o|
|x|o|x| 

X wins
</pre>

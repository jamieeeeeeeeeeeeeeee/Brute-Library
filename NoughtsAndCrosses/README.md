# Noughts And Crosses Brute Forcer
Exhaustive search for noughts and crosses (tic-tac-toe).

## Usage
`exhaust` for full search

`exhaust .........` for a specific position
(assumes X is the first player)
<br></br>
### Example
`exhaust ..o.xoxox`
<pre>
| | |o|
| |x|o|
|x|o|x| 

X wins
</pre>
